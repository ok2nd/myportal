「MyHome Portal」動作環境です。

動作環境

  • Webシステムとして動作します。
  • Webシステムといっても、サーバー用のコンピュータやサーバーOSが必要という訳ではありません。
    1台のパソコン上に、サーバー側のXAMPP+MyHome Portalとクライント側のブラウザを同居させて利用することができます。
    Windows XPベースの非力なパソコンでも十分に動作します。
  • 開発環境
  • （Ver.3.27以降）  XAMPP 1.7.1 (Windows)
  • + Apache 2.2.11
  • + MySQL 5.1.33 (Community Server)
  • + PHP 5.2.9 + PEA
  • （Ver.3.26以前）  XAMPP 1.6.6a (Windows)
  • + Apache 2.2.8
  • + MySQL 5.0.51a
  • + PHP 5.2.5
